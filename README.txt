数据可视化课程期末项目

项目题目：基于健康与生活方式数据的脑卒中风险可视化分析与预测系统

文件说明：
1. course_paper.docx：。
2. course_paper.pdf：
3. brain_stroke.csv：原始数据集。
4. analysis.py：数据清洗、统计分析、绘图和模型训练程序。
5. app.py：Streamlit交互式可视化程序。
6. requirements.txt：Python依赖。
7. START_WINDOWS.bat：Windows一键运行脚本。

Windows运行方法：
双击 START_WINDOWS.bat。首次运行会安装依赖并生成分析结果，完成后浏览器自动打开本地页面。
